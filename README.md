I am a developer focused on advanced **Unreal Engine systems engineering** and scalable gameplay architecture using **C++ and Blueprint integration**.

My primary interest lies in creating reusable systems that can scale cleanly across large projects while remaining modular, maintainable, and production-oriented.

Rather than only implementing gameplay features, I enjoy building the underlying infrastructure that powers those systems:

- gameplay frameworks
- runtime communication layers
- generic APIs
- serialization systems
- modular gameplay architecture
- plugin ecosystems
- data-driven pipelines
- editor/runtime tooling

I enjoy solving architectural and systems-level problems and turning complex workflows into reusable engineering patterns.
